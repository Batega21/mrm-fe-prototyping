1) Antes de comenzar a desarrollar, ejecutar el siguiente comando desde este directorio:
$> npm install
2) Finalizada la instalación de las dependencias, ejecutar el comando:
$> gulp serve

>>>>>>>>>>>>>>>   IMPORTANTE   <<<<<<<<<<<<<<<<<
> Hay que tener instalado Node.js, npm, y gulp.<
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>


Ubicación código fuente para subir a la webapp:
./build/index.html (Tanto JS como CSS se encuentra inline)
